# Madani Islamic Academy Management System

This is a demo academy management system created for learning purposes.

## Project Purpose

The purpose of this project is to understand how frontend, backend, and database work together in a simple full stack application.

## Tools Used

Python  
Streamlit  
SQLite  
Pandas  
Plotly  

## Features

Dashboard  
Student management  
Teacher management  
Fee management  
View and search records  
Charts and summary cards  

## How It Works

Streamlit is used for the frontend interface.  
Python functions are used for backend logic.  
SQLite is used to store data.  
Pandas is used to read and display data in table format.  
Plotly is used to show charts.

## How to Run

Install required libraries:

pip install streamlit pandas plotly

Run the app:

streamlit run app.py

## Important Note

This is a demo learning project. Real student data, parent phone numbers, fee records, and private academy information should not be uploaded to GitHub.
